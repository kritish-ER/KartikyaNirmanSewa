# 🏠 Dream Builders Website

A simple and modern **Builder / House Design Website** created using **HTML and CSS only**.
This project allows users to explore different house designs and get an idea based on their budget.

---

## 🚀 Features

* 🏡 Attractive homepage with hero section
* 🏘️ House design cards (2BHK, 3BHK, Villa, etc.)
* 💰 Budget planner (UI-based)
* 📞 Contact form
* 🎨 Responsive layout using CSS Grid
* ✨ Hover effects and smooth design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
